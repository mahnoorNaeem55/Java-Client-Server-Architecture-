# Java Networking & Algorithms Projects

This repository contains multiple Java projects developed as part of academic coursework and practice, focusing on networking concepts and algorithm implementation with graphical user interfaces.

## Projects Included
- **Client–Server Architecture (Socket Programming)**  
  Java-based client-server communication system using TCP sockets and GUI.

- **Prim’s Routing Algorithm (Graph Algorithm)**  
  Java implementation of Prim’s Minimum Spanning Tree algorithm with GUI-based visualization.
